Introdução

O site Sabor da Terra foi desenvolvido
com o objetivo de facilitar a compra de
produtos orgânicos, naturais e
artesanais de forma prática, organizada
e acessível aos clientes.
A plataforma apresenta um ambiente
intuitivo, contendo diversas
funcionalidades que auxiliam o usuário
durante toda a navegação, desde a
busca pelos produtos até a finalização
da compra.

Funcionalidades do site 

Ao acessar o site, o usuário encontra a
página inicial contendo os principais produtos
e categorias disponíveis na loja. O layout foi
planejado para proporcionar fácil navegação
e boa experiência visual.
Entre os principais recursos disponíveis estão:
- Barra de pesquisa;
- Ícone de sacola/carrinho;
- Botão para voltar ao início;
- Categorias de produtos;
- Área “Sobre Nós”;
- Espaço para dúvidas;
- Rodapé com informações de contato

Barra de Pesquisa

A barra de pesquisa permite que o usuário procure
produtos específicos de forma rápida e eficiente.
Basta digitar o nome do item desejado para
localizar os produtos disponíveis no catálogo.
Exemplos:
- frutas;
- verduras;
- queijos;
- cestas;
- geleias;
- pães artesanais.

Categorias de Produtos

O site possui uma divisão organizada por
categorias, facilitando a localização dos produtos.
Entre as sessões disponíveis estão:
- Verduras;
- Frutas;
- Laticínios;
- Produtos artesanais;
- Pães;
- Geleias;
- Cestas prontas.
Cada categoria apresenta produtos específicos
relacionados ao tema selecionado.

Cestas Prontas

Uma das funcionalidades do site é a venda de
cestas prontas. O usuário pode visualizar
diferentes opções de cestas organizadas por
tamanho e quantidade de produtos.
Além disso, cada cesta possui um botão de “Ver
detalhes”
, onde é possível consultar:
- os produtos inclusos;
- quantidade de itens;
- descrição da cesta;

Sacola

Existe o carrinho/sacola de compras, responsável
por armazenar os itens selecionados durante a
navegação.
Na etapa de finalização da compra, o cliente
encontra:
- resumo dos produtos;
- quantidade escolhida;
- valor total;
- opções de pagamento;
- continuar comprando ;
- finalizar compra.

Sobre Nós

A seção “Sobre Nós” apresenta a história da
empresa Sabor da Terra, explicando:
- a origem do negócio;
- os objetivos da empresa;
- o compromisso com produtos orgânicos e
naturais;
- a valorização dos produtos artesanais e
produtores locais.
Essa área ajuda a aproximar os clientes da
proposta da empresa.

Espaço de dúvidas

O site possui uma área destinada às dúvidas dos
usuários. Nesse espaço, os clientes podem:
- realizar perguntas;
- buscar informações;
- solicitar ajuda;
- tirar dúvidas sobre produtos ou pedidos.

Rodapé 

No rodapé do site encontram-se informações importantes para contato com a empresa,
como:
- telefone;
- e-mail;
- redes sociais;
- CEP.

Sistema de Login/Cadastro

O sistema conta com páginas separadas para
acesso dos usuários:
Página de Login
Destinada aos clientes que já possuem conta
cadastrada no sistema. O usuário informa:
- e-mail;
- senha.
Após o login, o cliente pode acessar sua conta
e continuar suas compras.

Página de Cadastro

Voltada para novos usuários que ainda não
possuem conta. Nessa página, o cliente realiza
o cadastro informando dados como:
- nome;
- e-mail;
- endereço;
- telefone;
- senha.
Após o cadastro, o usuário poderá utilizar
todas as funcionalidades disponíveis no site.

Conclusão

O site Sabor da Terra foi desenvolvido com
foco na praticidade, organização e facilidade
de navegação. As funcionalidades
implementadas permitem que os usuários
encontrem produtos rapidamente, realizem
compras com segurança e conheçam
melhor a proposta da empresa.
Além disso, o sistema de login, cadastro,
carrinho, e categorias organizadas contribui
para uma experiência mais completa e
eficiente ao cliente.
